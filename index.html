By logging into the FTP server, I was able to customize the CSS on a clientâ€™s website by using my own code editor and uploading the changes I made through the FTP server and onto the website. This was tricky because the website was built using Drupal 8 website content management system, so I had to find the correct class and ids to target by using Chrome DevTools.

Again, this was very tricky because of the hardcoded CSS in the Drupal webpage design. I had to rely heavily on Chrome DevTools to find the correct classes to target. I achieved the desired look by using the flex-wrap: wrap property and value along with overriding the Drupal code by changing the flex-basis with an appropriate property and value with !important; to override. I really enjoyed applying CSS to customize the precoded Drupal pages.
